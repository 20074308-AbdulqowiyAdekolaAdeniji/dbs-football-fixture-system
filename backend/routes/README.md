# API Routes

This folder contains route definitions for the API.

## Structure
- `fixtures.js` - Fixture-related endpoints (CRUD operations)
- `venues.js` - Venue-related endpoints (future)

## Endpoints

### Fixtures
- GET /api/fixtures - Get all fixtures
- GET /api/fixtures/:id - Get single fixture
- POST /api/fixtures - Create new fixture
- PUT /api/fixtures/:id - Update fixture
- DELETE /api/fixtures/:id - Delete fixture
